\chapter{量子网络物理建模}
\label{chap:phymodel}

本章建立多跳量子中继网络的物理模型，推导经典信令时延对端到端纠缠保真度的定量影响。这些结论是后续章节分析路由非保序性和设计保真度感知路由算法的理论基础。

\section{多跳量子中继网络的物理模型}
\label{sec:network-model}

\subsection{节点与链路参数}

考虑由$n$个量子中继节点$v_1, v_2, \ldots, v_n$构成的线性量子网络，其中$v_1$为源节点，$v_n$为目的节点。每个节点$v_i$配备量子存储器，其退相干参数（相干时间）记为$\tau_i > 0$。相邻节点$v_i$与$v_{i+1}$之间存在量子信道，可以概率性地生成Bell纠缠对；同时存在经典信道，用于传递测量结果等控制信令，经典信令从$v_i$传至$v_{i+1}$的传播时延记为$\delta_i \geq 0$。

在不失一般性的前提下，本章对单跳链路的纠缠生成保真度建模如下：链路$(v_i, v_{i+1})$生成的Bell对初始保真度记为$F^{(0)}_i \in (0, 1]$，该值由信道噪声（去极化噪声、相位翻转等）和纠缠生成操作的不完美性共同决定。在qns-3当前实现中，这一初始链路态以Werner态参数化：
\begin{equation}
  \rho_i^{(0)}
  = F_i^{(0)} \vert\Phi^+\rangle\langle\Phi^+\vert
  + \frac{1 - F_i^{(0)}}{3}
    \left(
      \vert\Phi^-\rangle\langle\Phi^-\vert +
      \vert\Psi^+\rangle\langle\Psi^+\vert +
      \vert\Psi^-\rangle\langle\Psi^-\vert
    \right)
  \label{eq:werner-init}
\end{equation}
其中$\vert\Phi^\pm\rangle,\vert\Psi^\pm\rangle$为四个Bell基态。

\subsection{纠缠交换协议的时序模型}

建立$v_1$到$v_n$端到端纠缠的标准协议是\emph{链式纠缠交换}：在$t=0$时刻，全部$n-1$条相邻链路同时尝试生成纠缠对；成功生成后，中间节点$v_k$（$k = 2, \ldots, n-1$）对其持有的来自$v_{k-1}$和$v_{k+1}$的两个量子比特执行贝尔态测量（Bell State Measurement, BSM），测量结果$\{m_L, m_R\} \in \{0,1\}^2$通过经典信道传递给下一节点$v_{k+1}$以指导纠正操作。

\textbf{关键时序约束：}量子比特在存储器中等待的时间直接决定退相干损耗。节点$v_k$的量子比特自$t=0$起等待，直至收到来自$v_{k-1}$的经典信令方可完成交换。由于信令沿链路逐跳传播，节点$v_k$在执行纠正操作时的等待时间$T(k)$满足如下递推关系：

\begin{equation}
  T(k) = \sum_{i=2}^{k-1} \delta_i, \quad k = 2, 3, \ldots, n-1
  \label{eq:waiting-time}
\end{equation}

即$v_k$的等待时间等于其左侧所有中继链路经典时延之和。特别地，$T(2) = 0$（第一个中继节点无需等待），$T(n-1) = \sum_{i=2}^{n-2} \delta_i$（最后一个中继节点等待时间最长）。

\section{端到端纠缠保真度的解析公式}
\label{sec:fidelity-formula}

\subsection{退相干模型}

qns-3当前的存储退相干模型并不直接把Bell保真度乘上一个指数因子，而是对\emph{单个存储量子比特}施加相位翻转信道
\begin{equation}
  \mathcal{E}_{\lambda}(\rho)
  = \frac{1 + \lambda}{2}\rho
  + \frac{1 - \lambda}{2}(Z\rho Z),
  \qquad
  \lambda(\Delta t, \tau) = e^{-\Delta t / \tau},
  \label{eq:phase-flip-memory}
\end{equation}
其中$\Delta t$为等待时间，$\tau$为该量子比特所在节点的相干时间。对一对以式~\eqref{eq:werner-init}初始化的Bell对，若左右两端量子比特分别等待$\Delta t_L,\Delta t_R$，则其状态仍保持Bell对角形式：
\begin{equation}
  \rho_i(\Delta t_L, \Delta t_R)
  = a_i \vert\Phi^+\rangle\langle\Phi^+\vert
  + b_i \vert\Phi^-\rangle\langle\Phi^-\vert
  + c_i \vert\Psi^+\rangle\langle\Psi^+\vert
  + d_i \vert\Psi^-\rangle\langle\Psi^-\vert,
  \label{eq:single-link-bell-diagonal}
\end{equation}
其中
\begin{align}
  q_i &\triangleq \frac{1 - F_i^{(0)}}{3}, \notag\\
  \Lambda_i &\triangleq
    \exp\!\left(-\frac{\Delta t_L}{\tau_i} - \frac{\Delta t_R}{\tau_{i+1}}\right), \notag\\
  a_i &= \frac{2F_i^{(0)} + 1 + \Lambda_i(4F_i^{(0)} - 1)}{6}, \notag\\
  b_i &= \frac{2F_i^{(0)} + 1 - \Lambda_i(4F_i^{(0)} - 1)}{6}, \notag\\
  c_i &= d_i = q_i. \label{eq:single-link-bell-weights}
\end{align}
因此，该链路在等待后的Bell保真度应写为
\begin{equation}
  F_i(\Delta t_L, \Delta t_R)
  = \langle\Phi^+\vert \rho_i(\Delta t_L, \Delta t_R) \vert\Phi^+\rangle
  = a_i,
  \label{eq:single-link-fidelity}
\end{equation}
而不是简单的$F_i^{(0)} e^{-\Delta t / \tau}$。

\subsection{多跳端到端保真度}

设Bell对角态的Bell权重向量记为
\begin{equation}
  w = (a, b, c, d),
  \qquad a + b + c + d = 1,
\end{equation}
其中$a$即相对于$\vert\Phi^+\rangle$的Bell保真度。对两端等待导致的相位翻转退相干，可定义等待算子
\begin{equation}
  \mathcal{M}_{\Lambda}(a, b, c, d)
  =
  \left(
    \frac{1 + \Lambda}{2}a + \frac{1 - \Lambda}{2}b,\;
    \frac{1 - \Lambda}{2}a + \frac{1 + \Lambda}{2}b,\;
    \frac{1 + \Lambda}{2}c + \frac{1 - \Lambda}{2}d,\;
    \frac{1 - \Lambda}{2}c + \frac{1 + \Lambda}{2}d
  \right),
  \label{eq:bell-memory-operator}
\end{equation}
其中$\Lambda$由式~\eqref{eq:single-link-bell-weights}给出。

若两段相邻路径的端到端态分别为$w=(a,b,c,d)$与$w'=(a',b',c',d')$，则一次理想纠缠交换后的输出仍为Bell对角态，其Bell权重满足
\begin{equation}
  \mathcal{S}(w, w')
  =
  \bigl(
    aa' + bb' + cc' + dd',\;
    ab' + ba' + cd' + dc',\;
    ac' + ca' + bd' + db',\;
    ad' + da' + bc' + cb'
  \bigr).
  \label{eq:bell-swap-operator}
\end{equation}
因此，多跳端到端纠缠保真度的\textbf{正确形式}不是标量乘积，而是对各跳Bell对角态先施加等待算子$\mathcal{M}_{\Lambda}$，再递归施加交换算子$\mathcal{S}$；最终端到端保真度等于递推结果的第一个分量。

特别地，在\emph{无额外等待}且输入均为Werner态的情形，若两条相邻链路初始保真度分别为$F_1,F_2$，则一次纠缠交换后的Bell保真度为
\begin{equation}
  F_{\mathrm{swap}}
  = F_1 F_2 + \frac{(1 - F_1)(1 - F_2)}{3}
  = \frac{4F_1F_2 - F_1 - F_2 + 1}{3},
  \label{eq:werner-swap-fidelity}
\end{equation}
而不是简单的$F_1F_2$。式~\eqref{eq:single-link-fidelity}和式~\eqref{eq:werner-swap-fidelity}正是当前qns-3物理层混态建模与理想纠缠交换所对应的闭式表达。

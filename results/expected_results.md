# 四节点量子链协议实验预期结果

## 实验参数

- **链路保真度 (LinkF)**: 0.85, 0.90, 0.95, 0.99
- **T2退相干时间**: 10ms, 50ms, 100ms, 500ms  
- **经典通信延迟**: 1ms, 5ms, 10ms, 50ms
- **总协议时间**: 3 × 延迟 (Phase1→BSM_B→BSM_C→Corrections)

## 理论保真度计算公式

```
F_final = F_link³ × F_decoherence
F_decoherence = (1 + exp(-2t/T2)) / 2
```

其中 t = 3 × delay 为总协议时间。

## 关键发现

1. **链路质量是主导因素**: 链路保真度从0.85提升到0.99，平均最终保真度提升58%

2. **T2/延迟比至关重要**: 
   - T2=500ms, delay=1ms (比值500): F可达0.96
   - T2=10ms, delay=50ms (比值0.2): F仅为0.31-0.49

3. **实用性阈值**:
   - F > 0.9: 需要 LinkF≥0.99, T2≥100ms, delay≤5ms
   - F > 0.8: 需要 LinkF≥0.95, T2≥50ms, delay≤5ms
   - F > 0.7: 需要 LinkF≥0.90, T2≥100ms, delay≤5ms

4. **退相干测试验证**: 5/5测试通过，QuantumMemoryModel正确实现T1/T2退相干
